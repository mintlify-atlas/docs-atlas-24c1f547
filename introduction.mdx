# Unikraft CLI

The official command-line interface for [Unikraft Cloud](https://unikraft.cloud) — deploy and manage unikernels globally in milliseconds.

<Note>
This is the new Unikraft CLI which will eventually supersede `kraft cloud`. It is still in early development and subject to change before the v1.0.0 release. [Feedback appreciated](https://unikraft.link/devsurvey)!
</Note>

## What is Unikraft?

Unikraft is a fast, secure and open-source unikernel development kit. Unikernels are specialized, single-address-space machine images that boot in milliseconds and have minimal attack surface. The Unikraft CLI lets you deploy these unikernels as serverless instances on Unikraft Cloud's global infrastructure.

## Quick Example

Deploy an NGINX instance in Frankfurt with HTTPS enabled:

```bash
unikraft run --metro=fra -p 443:8080/http+tls nginx:latest
```

This single command:
- Deploys a unikernel instance in the Frankfurt metro
- Exposes port 8080 as HTTPS on port 443
- Automatically provisions TLS certificates
- Returns the instance URL in milliseconds

## Why Unikraft CLI?

**Fast Boot Times** — Unikernels boot in milliseconds, not seconds. Perfect for serverless workloads.

**Global Reach** — Deploy across multiple geographic regions with a single command.

**Cost Efficient** — Scale-to-zero support means you only pay for what you use.

**Developer Friendly** — Familiar CLI patterns, multiple output formats, and shell completions.

**Production Ready** — Full resource lifecycle management with comprehensive monitoring and logging.
